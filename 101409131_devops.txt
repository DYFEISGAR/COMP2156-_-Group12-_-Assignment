The course "COMP2156 - DevOps For System Administration" is designed to teach students the essential procedures, protocols, and best practices in DevOps to enhance productivity in workplace environments. It covers a range of topics including data formats (XML, JSON, YAML, YANG), DevOps tools (Ansible, Git), and techniques for implementing DevOps solutions to complex business problems. Students will learn through lectures, demonstrations, hands-on exercises, and assignments, with a focus on using industry-standard tools for system or network engineering within DevOps processes. The course aims to equip students with the skills to effectively carry out DevOps roles, utilize version control systems, make architectural decisions, monitor code quality, and deploy software products using cloud computing and Infrastructure as Code.
